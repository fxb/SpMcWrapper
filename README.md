# SpMcWrapper

A Java Minecraft wrapper which sits in between the official Minecraft client and server.

## Usage

Put it into the same directory as *minecraft-server.jar* and edit *settings.json* to suit your needs.

## Dependencies

* Google Gson (*gson-1.5.jar*) - http://code.google.com/p/google-gson/
